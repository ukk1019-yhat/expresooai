Behavioral Intelligence API

Integrate scoring, simulation logic, and conversational analytics directly into your own product or LMS.

Initialize a Session

Start a new simulation session by defining the persona ID and the difficulty parameters.

POST /v1/sessions/create

The API returns a secure WebSocket URL for real-time bidirectional audio streaming.

curl -X POST https://api.expresso.ai/v1/sessions/create \ -H "Authorization: Bearer sk_test_12345" \ -H "Content-Type: application/json" \ -d '{ "persona_id": "pr_negotiation_hard", "user_id": "usr_9982", "options": { "stream_audio": true, "record_session": true } }' // Response { "session_id": "ses_001239", "websocket_url": "wss://stream.expresso.ai/ses_001239", "status": "ready" }
