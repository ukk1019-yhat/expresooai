## в—Ҳ What Is EXPRESSO AI?

> Most platforms teach you **what** to say.  
> EXPRESSO AI teaches you **how** to be heard.

EXPRESSO AI is a **Behavioral Intelligence Infrastructure** вҖ” a simulation engine that recreates the cognitive and emotional complexity of real human interactions. Instead of static courses, you're dropped into live AI conversations with dynamic personas: a skeptical investor, a difficult team member, a high-stakes negotiator.

Every word you say is **scored in real-time**. Every emotional shift is **tracked and analyzed**. Every session ends with a behavioral report that shows you exactly where you succeeded, where you hesitated, and where you lost the room.

- **AI Layer** вҖ” LLM-powered persona simulation & behavioral scoring
- **Deployment** вҖ” Vercel Edge Network
- **Extension** вҖ” Chromium-compatible extension (`/extension`)
- **Languages** вҖ” TypeScript `89%` В· JavaScript `9%` В· HTML/CSS `2%`

## в—Ҳ Quick Start

**Prerequisites:** Node.js 18+, a Supabase project, a Clerk application

```bash
# Clone the repository
git clone https://github.com/ukk1019-yhat/expresooai.git
cd expresooai

# Install dependencies
npm install

# Configure environment
cp .env.example .env.local
# вҶ’ Add your NEXT_PUBLIC_CLERK_PUBLISHABLE_KEY
# вҶ’ Add your CLERK_SECRET_KEY
# вҶ’ Add your NEXT_PUBLIC_SUPABASE_URL
# вҶ’ Add your NEXT_PUBLIC_SUPABASE_ANON_KEY

# Initialize the database
# вҶ’ Run supabase-schema.sql in your Supabase SQL editor

# Start the development server
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) вҖ” your behavioral intelligence platform is live.

## в—Ҳ Environment Variables

| Variable | Description |
|---|---|
| `NEXT_PUBLIC_CLERK_PUBLISHABLE_KEY` | Clerk public key (frontend auth) |
| `CLERK_SECRET_KEY` | Clerk secret key (server-side auth) |
| `NEXT_PUBLIC_SUPABASE_URL` | Your Supabase project URL |
| `NEXT_PUBLIC_SUPABASE_ANON_KEY` | Supabase anonymous key |

> вҡ пёҸ Never commit `.env` to version control. The `.gitignore` covers this, but double-check before pushing.

The `/extension` directory contains the EXPRESSO AI Chrome extension вҖ” bringing behavioral intelligence directly into any web conversation.

```bash
2. Enable "Developer Mode" (top right)

4. Select the /extension folder
```

## в—Ҳ Database

The full Supabase schema lives in `supabase-schema.sql`. It defines the data model for users, sessions, personas, behavioral scores, and analytics events.

To initialize:
1. Open your Supabase dashboard вҶ’ SQL Editor
2. Paste and run `supabase-schema.sql`
3. Verify tables are created under the `public` schema

## в—Ҳ The Philosophy

Most communication training gives you scripts. Scripts break the moment reality diverges from the expected.

EXPRESSO AI takes a different approach вҖ” it trains the **behavioral layer beneath the words**: how you respond under pressure, how you read emotional shifts, how you adapt when someone pushes back harder than expected.

The goal isn't to make you sound better. It's to make you think faster, feel more clearly, and communicate with precision when it actually matters.
